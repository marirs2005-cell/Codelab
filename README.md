# Proyecto: Generación de Matriz RGB a partir de una Imagen

# Descripción del proyecto
Este proyecto consiste en analizar cómo se representan las imágenes digitales a partir de datos numéricos. Para ello, se seleccionó una imagen de interés personal (un gato) y se transformó en información estructurada dentro de Excel.

A partir de esta imagen, se extrajeron los valores RGB (Red, Green, Blue) de cada píxel con el fin de construir una matriz de colores, lo que permite comprender cómo las imágenes digitales pueden representarse mediante datos y matrices numéricas.

# Objetivos

# Objetivo general:
Comprender la representación digital de una imagen mediante la construcción de su matriz RGB.

# Objetivos específicos:

* Seleccionar una imagen de interés personal.

* Convertir la imagen en datos procesables.

* Extraer los valores de color RGB de cada píxel.

* Organizar la información en Excel.

* Analizar cómo una imagen puede representarse como una matriz de datos.


# Imagen utilizada

La imagen seleccionada corresponde a un gato, elegido como ejemplo para el desarrollo del ejercicio. Las imágenes digitales están formadas por píxeles, y cada píxel contiene tres componentes de color:

🔴 Rojo (Red)

🟢 Verde (Green)

🔵 Azul (Blue)

Cada uno de estos valores puede tomar números entre 0 y 255, lo que determina la intensidad del color.


# Metodología

Para desarrollar el proyecto se siguieron los siguientes pasos:

1. Selección de la imagen: Se eligió una imagen de un gato como base del ejercicio.

2. Conversión de la imagen a datos: La imagen fue transformada en un formato que permitiera visualizar los valores de los píxeles.

3. Extracción de valores RGB: Se identificaron los valores de Red, Green y Blue de cada píxel de la imagen.

4. Organización en Excel: Los datos fueron organizados en una hoja de cálculo, estructurando la información en forma de tabla.

5. Construcción de la matriz RGB: Finalmente, los valores obtenidos se organizaron en una matriz que representa la composición de colores de la imagen.

# Aprendizajes obtenidos

Este proyecto permitió comprender:

* Cómo se representan las imágenes digitalmente.

* La estructura del modelo de color RGB.

* Cómo los píxeles pueden convertirse en datos numéricos.

* La relación entre imágenes, matrices y análisis de datos.
