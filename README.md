Para el desarrollo de este proyecto se seleccionó una imagen de interés personal (en este caso un gato) y se transformó en datos numéricos para analizar cómo se representa digitalmente. La imagen fue convertida a formato Excel para obtener y organizar los valores RGB (Red, Green, Blue) de cada píxel.

El objetivo de la activdad es comprender cómo una imagen digital se compone de píxeles y cómo cada píxel puede representarse mediante una matriz de valores RGB (0–255).

Proceso:
* Seleccionar la imagen
* Conversión de la imagen a datos en Excel
* Extracción de los valores RGB de cada píxel
* Construcción de la matriz RGB
